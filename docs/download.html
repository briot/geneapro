Installing on Windows

This application depends on several external tools that you will need to install first. We plan on making things easier ultimately, but for now you'll have to install each of these individually.
  1. Download and install the Windows python installer for your machine (either 32 bits or 64 bits).
  2. Download and install the precompiled Sqlite for Windows. I think you only need the DLL version, not the shell command line.
  3. Download and install the latest official release of Django. You will need 7-zip or Winzip to unpack the archive, and then you will need to run the python command listed on the download page.
  4. Finally, once these are working, you can download Geneaprove and launch it.
